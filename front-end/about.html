About

This project was developed as a Database management project by
Kaustubh, Projjal, Sayan, Sourav and Aditya.

This website provides user and restaurant sign up.
The user can see various menus provided by the restaurants and order anything.

Once a restaurant signs up, they have the option to update their own menu.
Customer orders shows up on the dashboard, and can be ticked off once delivered.

This website is a stand-alone complete product, and can be used commercially
once some payment authentication is set up for customer order.

The source code is available on GitHub
